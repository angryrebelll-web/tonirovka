# Проверка калькулятора на GitHub Pages

## Проблема
Калькулятор показывает 404 ошибку при открытии.

## Что проверено
1. ✅ Все файлы закоммичены в репозиторий
2. ✅ Файл .nojekyll создан
3. ✅ Пути исправлены (относительные без префиксов)
4. ✅ Base href установлен на полный URL

## Файлы в репозитории
- calculator.html
- calculator-fullscreen.css
- js/calculator-fullscreen.js
- js/cars.js
- js/prices.js
- icons/*.svg (7 файлов)
- .nojekyll

## Что нужно проверить на GitHub

1. **Настройки GitHub Pages:**
   - Перейдите: https://github.com/angryrebelll-web/landpadge/settings/pages
   - Source: Deploy from a branch
   - Branch: master / (root)
   - Save

2. **Проверка деплоя:**
   - Перейдите: https://github.com/angryrebelll-web/landpadge/actions
   - Убедитесь, что последний workflow завершился успешно

3. **Прямые ссылки для проверки:**
   - https://angryrebelll-web.github.io/landpadge/calculator.html
   - https://angryrebelll-web.github.io/landpadge/calculator-fullscreen.css
   - https://angryrebelll-web.github.io/landpadge/js/calculator-fullscreen.js

## Если прямые ссылки не работают
- Проверьте, что файлы действительно в репозитории
- Проверьте настройки GitHub Pages
- Подождите 5-10 минут после последнего коммита

## Если прямые ссылки работают, а iframe нет
- Возможно проблема в CORS или безопасности iframe
- Попробуйте использовать полный URL в src iframe
